# QuantumManyBody_FromZero
Personal learning code. Quantum many-body numerical algorithms (MeanField, ED, DMRG, etc.) with detailed comments (in Chinese) for beginners. Reimplementation of classic results and diagrams.

## ⚠️ Note on viewing Jupyter notebooks

GitHub has known issues rendering `.ipynb` files and may show `An error occurred` (e.g., using nbformat v5.10.4 and nbconvert v7.17.1).  
If a notebook fails to load, please copy its URL and paste it into [nbviewer.org](https://nbviewer.org) – it will render the notebook correctly.
